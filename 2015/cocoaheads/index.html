Então você quer desenvolver um app nativo?

...

Mais plataformas, mais problemas :/

Problemas?

  • Proficiência em cada plataforma necessária
  • Bases de código claramente diferentes e separadas
  • Processo de desenvolvimento requer mais tempo e dinheiro
  • Menos retorno :/

Então os SDKs nativos são ruins?

NÃO!

Eles são incríveis <3

  • UI comum
  • Views
  • Navegação e Histórico no Stack
  • Transições
  • Gestos
  • ...

Não dá pra melhorar?

Como começou?

  • 2009: Projeto Phonegap iniciado na Nitobi
  • 2011: Adobe compra o Phonegap
  • 2012: Phonegap doado a Apache com o nome de Cordova
  • Agora: Phonegap é uma distribução de Cordova mantida pela Adobe

Mas o que é?

  • Web abstraída em camada nativa
  • Acesso direto a APIs nativas
  • Ambiente de desenvolvimento familiar para desenvolvedores web
  • Desenvolver uma única base de código (web)

Mas... aplicações híbridas são lentas!

Os tempos são outros ;)

Os dispositivos móveis têm melhorado ;)

E onde está o SDK da web?

  • Necessidade de preencher a lacuna entre web e nativo
  • Necessidade de componentes de UI e interações ricas e nativas
  • Necessidade de APIs de UI, e não apenas os widgets do jQuery

Ferramentas

Ionic CLI

$ npm install -g ionic

Facilita o processo de criação/execução/compilação/depuração de aplicações Ionic 

$ ionic start myapp [template]
$ ionic setup sass
$ ionic serve [options]
$ ionic platform [OS]
$ ionic run ios [options]
$ ionic resources [options]
$ ionic build [OS]

Ionic Playground

Demos no navegador, sem ter que instalar nada :)

play.ionic.io

Ionic Creator

Ferramenta drag-and-drop para prototipagem de interfaces

creator.ionic.io

Ionic Lab

Debug Android e iOS simultaneamente

$ ionic serve --lab

Ionic View

Visualizar/Compatilhar/Testar apps em vários aparelhos

$ ionic start myApp
$ cd myApp
$ ionic upload

Como começar?

  • Instalar homebrew e o Node.js
  • Instalar Cordova, Ionic e Gulp como dependências globais do Node.js
  • Google Chrome (para depuração)
$ brew install node
$ npm install -g cordova ionic gulp
