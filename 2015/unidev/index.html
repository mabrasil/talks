Pure Functions

--| Exemplos

quadrado :: Integer -> Integer
quadrado x = x * x

somaTres :: Integer -> Integer -> Integer -> Integer

Pattern Matching

checaIntervalo :: (Integral a) => a -> String
checaIntervalo 1 = "Um"
checaIntervalo 2 = "Dois"
checaIntervalo 3 = "Três"
checaIntervalo 4 = "Quatro"
checaIntervalo 5 = "Cinco"
checaIntervalo x = "Não tá no intervalo"

List Comprehensions

quadrado = [ x*x | x <- nums, x > 2 ]

-- | Se lembram do map?
[x+3 | x <- [1,5,3,1,6]]

-- | Com um predicado...
[ 2*x | x <- [0..], x^2 > 3 ]

Concorrência

Concorrência...

  • X tarefas em um mesmo período de tempo
  • Compartilham os recursos do sistema
  • Logicamente independentes

Concorrência...

✘ Identificar aspectos concorrentes no problema a ser resolvido

✘ Gerenciar como os dados são acessados e compartilhados entre as tarefas

Paralelismo...

  • Divisão de uma tarefa complexa em tarefas menores
  • Processadas de maneira independente e simultânea em múltiplas unidades de processamento

Paralelismo...

✘ Mudanças no estado tem de ser conhecidas e pensadas

Em resumo...

  • Paralelismo diz respeito a tarefas que rodam ao mesmo tempo, em múltiplos hardwares.
  • Concorrência diz respeito a tarefas que podem rodar independentemente, compartilhando hardware.
