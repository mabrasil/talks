#### [IX Pylestras](http://pylestras.org/evento/ix-pylestras/)

![Status](https://img.shields.io/badge/type-talk-brightgreen.svg?style=flat-square)

- **When:** May, 30.
- **Where:** Universidade de Fortaleza - Unifor.
- **Attendees:** ≈40.

- **When:** February, 27.
- **Where:** Universidade de Fortaleza - Unifor.
